## Local backend setup

1. Install Node.js from https://nodejs.org if it is not already installed.
2. Open a terminal in `c:\Users\mutee\project`.
3. Run `npm install`.
4. Run `npm start`.
5. Open `http://localhost:3000/products.html` or `http://localhost:3000/cart.html` in your browser.

## Notes

- This project uses a simple Express backend to serve product data and mock checkout flow.
- Real mobile money integration requires a payment provider API.
